### 擴充短篇文章

**工作要點**

以下文章內容好但篇幅過短，需要擴充：

1. **季節之間**（507 字 → 目標 1,500-2,000 字）
   - 每個季節的描寫可以更細膩
   - 「年輪」的意象可以貫穿更多
   - 可加入更多「在日本感受季節」的具體場景

2. **遷徙**（475 字 → 目標 1,500-2,000 字）
   - 「十八年後引用同一首詩」是很好的時間折疊
   - 可以擴充：當年為什麼引用？現在的處境有何不同？

3. **城市燈光熄滅與日出之前的縫隙**（162 字）+ **清晨傳來的溫度**（360 字）
   - 考慮合併為「伊斯坦堡的晨光」
   - 擴充城市的感官描寫（聲音、氣味、溫度）
   - 流浪狗的陪伴可以成為情感核心

4. **材質背後的模糊光線**（155 字 → 目標 1,000-1,500 字）

### 決定編排結構與輯名

**工作要點**

待文章數量累積到一定程度後進行：

1. **重新審視五輯架構**
   - 輯一：時間的皺褶
   - 輯二：異鄉的縫隙
   - 輯三：光的切片
   - 輯四：連結與孤獨
   - 輯五：答案之書

2. **調整輯名**
   - 是否都要呼應「折疊」意象？
   - 輯名是否足夠詩意但不晦澀？

3. **決定每輯的文章配置**
   - 長短交錯的節奏
   - 情緒的起伏安排
   - 每輯 4-6 篇為宜

4. **考慮是否需要**
   - 序言（自序或他序）
   - 後記
   - 輯間的過渡文字

5. **書名的方向**
   - 與「折疊」相關的意象
   - 簡潔但有餘韻

### 整體審閱與定稿

**工作要點**

這是最後階段，待所有文章完成初稿後進行：

1. **通讀全書**
   - 檢查主題一致性
   - 確認「折疊」意象的貫穿
   - 調整重複的意象或用語

2. **個別文章的最終潤飾**
   - 開頭與結尾的力道
   - 轉折處的流暢度
   - 刪除冗贅的段落

3. **整體節奏調整**
   - 確認長短交錯
   - 調整輯與輯之間的過渡
   - 確保閱讀不疲乏

4. **移除部落格痕跡**
   - 連結、hashtag、emoji
   - 過於口語或網路用語
   - 時效性過強的內容

5. **最終字數確認**
   - 是否達到 70,000 字目標
   - 每篇的篇幅是否平衡

- 所有完成的散文稿件

## 創作參考資訊

### 來自研究文件的洞察

> **文件路徑：** `docs/research/2025-12-23-blog-to-essay-collection.md`

**散文的核心特質**：
- 真實性：內容環繞作者的生命歷程與生活體驗
- 個人風格：融入作者的人格個性與情緒感懷
- 思想性：訴諸作者的觀照思索與學識智慧
- 文學性：在主題、意象、境界、布局、修辭、氣勢上有所經營

**你的寫作已具備的特質**：
- 生活觀察的敏銳度
- 意象的運用（光線、海浪、畫框、年輪）
- 情感的節制與留白
- 思想的深度
- 個人聲音的確立

**需要調整的面向**：
- 篇幅的一致性
- 主題的串聯
- 網路元素的處理
- 自我重複的篩選

### 來自 PRD 的創作方向

**核心意象：折疊**
- 時間的折疊：過去與現在的交錯
- 空間的折疊：異鄉與故鄉
- 記憶的折疊：層層疊加的連結
- 正反面的折疊：孤獨與連結、恐懼與好奇

**風格參考**
- 《入境大廳》：平實溫柔的筆調，從適應陌生到將他鄉化為日常
- 《東京模樣》：短篇故事結構、前後呼應的收尾
- 《我的完美日常》：在日常重複中發現美好

### 關鍵創作原則

1. **有靈感就寫**：不強求完整，先累積再整理
2. **草稿思維**：寫出來的不是完成品，允許反覆修訂
3. **折疊意識**：時時檢視是否呼應核心主題
4. **節制與留白**：不過度抒情，讓讀者自行感受
5. **從具體到抽象**：先有場景，再有哲思
