http://code.google.com/p/zen-coding/  
  
他有 vim 外掛，裝上去後，開啟你的 XML/HTML。如果你想要以下結構  
```
<div class="main">
  <p></p>
  <p></p>
  <p></p>
<div>

```  
只要打  
```
div.main>p\*3

```按下 Ctrl + Y，再按下逗點就可以展開成上面的語法了。
